## 最初に
このMODは歳を取らなくなる"不老"特性を追加します。  
キャラ生成時に不老特性が付く可能性は0.01%です。  
[EdB Prepare Carefully]等を使って手動で特性を付与する必要があります。  

Rimworld 1.0 を対象としています。  

## MODの読み込みの順番(Load order)
- Core  
- HugsLib  
- EternalYouthTraits
